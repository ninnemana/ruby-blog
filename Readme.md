Modular Sinatra Boilerplate:
 ~ Google Oauth2 Mixin
 ~ Mongoid/Mongo
 ~ RSpec
 ~ Compass/SASS
 ~ Reloader

For ruby-1.9.3-p194

Startup using 'foreman start' in root project folder. Any changes in 'sass' folder will be  
compiled into the screen.css file. 

Add new routes to collections/my_routes.rb
